# Resources: The Rendered-Resource Round-Trip

> Status: stub — owned by HOL-1504.

This chapter specifies the rendered-resource round-trip: how manifests
generated by tasks (Helm, Kustomize, files, commands) are lifted back into
CUE as typed values so that platform-wide constraints, policy mixins, and
transformers unify over them before final artifacts are written.  It defines
the resource addressing scheme (group/version/kind/namespace/name keys as
structs, not lists), the fidelity requirements of the round-trip, and the
error semantics when unification over generated output conflicts with a
component's declared configuration.
